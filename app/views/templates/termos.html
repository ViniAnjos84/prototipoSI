Termos de Uso e Política de Privacidade

Sistema SAECA - Sistema de Agendamentos e Eventos e Consultas Acadêmicas

Ao utilizar a plataforma SAECA, o usuário declara que leu, compreendeu e concorda com os presentes Termos de Uso e Política de Privacidade, autorizando o tratamento de seus dados pessoais conforme a Lei Geral de Proteção de Dados (LGPD - Lei nº 13.709/2018).

1. Dados Coletados

O sistema poderá coletar os seguintes dados pessoais:

  • Nome completo
  • CPF
  • Data de nascimento
  • E-mail
  • Telefone
  • CEP

Em casos específicos, também poderão ser coletados:

  • Dados do responsável legal de menores de idade
  • Dados relacionados ao PET cadastrado
  • Informações necessárias para agendamentos e atendimentos

2. Finalidade do Tratamento dos Dados

Os dados coletados possuem as seguintes finalidades:

  • Identificação segura do usuário
  • Gerenciamento de atendimentos e agendamentos
  • Comunicação e envio de notificações
  • Prestação adequada dos serviços oferecidos
  • Prevenção de fraudes e validação de identidade
  • Identificação e histórico de PETs
  • Cumprimento de obrigações legais e regulatórias
  • Melhoria contínua da plataforma

3. Consentimento do Usuário

O consentimento será solicitado de forma livre, informada e inequívoca durante o processo de cadastro, através de checkbox de aceite.

O sistema registrará:

  • Data e hora do aceite
  • Versão do termo aceito
  • Identificação do usuário

4. Segurança da Informação

O SAECA adota medidas técnicas e administrativas para proteger os dados pessoais armazenados.

  • Criptografia
  • Autenticação segura
  • Controle de sessões
  • Logout com invalidação de sessão
  • Consultas
  • Proteção contra ataques de força bruta
  • Uso de variáveis de ambiente para dados sensíveis

5. Compartilhamento de Dados

Os dados poderão ser compartilhados somente quando necessário:

  • Com profissionais responsáveis pelos atendimentos
  • Com sistemas essenciais para operação da plataforma
  • Mediante obrigação legal ou ordem judicial

6. Direitos do Titular dos Dados

O usuário poderá, a qualquer momento:

  • Solicitar acesso aos dados pessoais
  • Solicitar correção de informações incorretas
  • Solicitar exportação dos dados
  • Solicitar exclusão dos dados pessoais
  • Revogar o consentimento fornecido

7. Armazenamento dos Dados

Os dados serão armazenados em ambiente seguro pelo tempo necessário para cumprimento das finalidades previstas neste documento e das obrigações legais aplicáveis.

8. Responsabilidades do Usuário

O usuário é responsável por:

  • Fornecer informações verdadeiras e atualizadas
  • Manter a confidencialidade de suas credenciais
  • Não compartilhar acesso com terceiros
  • Utilizar o sistema de forma ética e legal

9. Alterações nos Termos

O SAECA poderá atualizar este documento sempre que necessário para adequação legal, melhorias de segurança ou atualização dos serviços.

Novas versões poderão exigir novo consentimento do usuário.

10. Conformidade com a LGPD

O tratamento dos dados pessoais realizado pelo SAECA segue os princípios previstos na Lei Geral de Proteção de Dados (LGPD), incluindo:

  • Finalidade
  • Necessidade
  • Transparência
  • Segurança
  • Prevenção
  • Responsabilização
